# German B2 Beruf Trainer

## 📖 О проекте
Локальный веб-тренажёр для изучения немецких слов (уровень B2 Beruf). Работает полностью в браузере, без сервера и баз данных. Данные хранятся в JSON файлах, прогресс — в localStorage.

**Особенности:**
- 🎴 Четыре режима обучения: карточки, тест, режим ввода, презентация
- 🏠 Современный дашборд со статистикой прогресса
- 🔊 Озвучивание слов (Web Speech API)
- 📊 Отслеживание прогресса по каждому слову
- 🔍 Поиск и фильтрация
- 📱 Полностью адаптивный дизайн
- 🎨 Плавные анимации и современный интерфейс

## 🛠 Технологии
- **HTML5** — структура
- **CSS3** — стилизация с градиентами и анимациями
- **Vanilla JavaScript** — вся логика
- **LocalStorage** — сохранение прогресса
- **Web Speech API** — озвучивание текста

### Основные функции

#### Управление данными
- `loadWords()` — загрузка JSON из data/ по выбранному дню, перемешивание слов
- `saveProgress(day, words)` — сохранение статусов слов в localStorage
- `loadProgress(day, words)` — загрузка сохранённого прогресса
- `updateDashboardStats()` — обновление статистики на главной странице

#### Навигация
- `showScreen(screenName)` — переключение между дашбордом и режимами
- Возврат на главную через кнопки "Назад"

#### Карточки
- `showCard()` — отображение текущей карточки (слово, перевод, примеры)
- `markWord(status)` — отметка "знаю/не знаю", автосохранение
- `showNextCard()` / `showPrevCard()` — навигация по карточкам

#### Режим ввода
- `toggleTypingMode()` — включение/выключение режима ручного ввода
- `checkTypingAnswer()` — проверка введённого перевода
- `prepareTypingMode()` — подготовка правильного ответа для текущего слова

#### Тест
- `createTestQuestions()` — генерация вопросов с 4 вариантами ответов
- `showTestQuestion()` — отображение текущего вопроса
- `handleTestAnswer()` — обработка выбранного ответа
- `showTestResult()` — отображение результата теста

#### Режим презентации
- `startPresentationMode()` — запуск полноэкранного режима презентации
- `showPresentationSlide()` — отображение текущего слайда с анимацией
- `startProgressBar()` — анимация прогресс-бара (requestAnimationFrame)
- `pausePresentation()` — пауза/возобновление презентации
- `exitPresentation()` — выход из презентации
- `updateWordsList()` — обновление боковой панели со списком слов
- `toggleSidebar()` — скрытие/показ боковой панели

#### Аудио
- `speakText(text, lang)` — озвучивание текста (de-DE / ru-RU)
- `speakCardWord()` — озвучивание текущего слова в карточках
- `speakPresentationWord()` — озвучивание слова в презентации
- `speakTestWord()` — озвучивание слова в тесте
- `toggleSound()` — включение/выключение звука

#### Поиск и фильтрация
- `repeatUnknownWords()` — показать только слова со статусом "не знаю"
- `showRandomWord()` — переход к случайному слову

## 🎯 Режимы обучения

### Главная страница (Дашборд) 🏠
Современный интерфейс с:
- **Статистика прогресса**: общее количество слов, изученные, в процессе, процент
- **Выбор набора слов**: Tag 1, Tag 2, Tag 3 с описаниями
- **Карточки режимов**: 4 режима с иконками, описаниями и списком функций
- **Дополнительные функции**: кнопки для сложных слов, случайного слова и управления звуком

### 1. Карточки 📇
Классический режим с карточками:
- Два направления: DE → RU и RU → DE
- Кнопки "Знаю" и "Не знаю" для оценки
- Автопереход к следующей карточке (опционально)
- Отображение примеров использования
- Кнопка озвучивания слова

### 2. Режим ввода ✏️
Режим активного запоминания (доступен внутри карточек):
- Пользователь вводит перевод в текстовое поле
- Автоматическая проверка (без учёта регистра)
- Автоматическая отметка "знаю" при правильном ответе
- Подсказка правильного ответа при ошибке

### 3. Тест 📝
Проверка знаний:
- 4 варианта ответа (1 правильный, 3 случайных)
- Визуальная индикация правильного/неправильного ответа
- Отображение результата в конце (X из Y, процент)
- Кнопка озвучивания слова

### 4. Режим презентации 🎬
Автоматический показ слов:
- Крупные шрифты (слово 96px, перевод 72px, примеры 32px)
- Плавное появление элементов (opacity)
- 11 секунд на слово
- Прогресс-бар внизу экрана
- Боковая панель со списком всех слов (можно скрыть кнопкой 📚)
- Управление: пауза, остановка, выход на главную
- Автоматическое озвучивание слова
- Кнопки с тёмным фоном для лучшей видимости

## 🔧 Работа с localStorage

### Ключи
- `germanTrainerProgress` — прогресс изучения слов

### Формат прогресса
```javascript
{
  "tag1": {
    "gelingen": "known",
    "abraten": "unknown",
    "antworten": null
  }
}
```

### Статусы
- `"known"` — слово отмечено как "знаю"
- `"unknown"` — слово отмечено как "не знаю"
- `null` — слово ещё не оценено

## 🚀 Как добавить новый набор слов

1. Создать файл `data/tag4.json`
2. Добавить в `index.html` новую кнопку в `.day-buttons`:
   ```html
   <button class="day-btn" data-day="tag4">Tag 4 <span>Описание набора</span></button>
   ```
3. Добавить слова в формате:
   ```json
   [
     {
       "word": "слово",
       "translation": "перевод",
       "example_de": "пример на немецком",
       "example_ru": "пример на русском"
     }
   ]
   ```

## 🔄 Рабочий процесс для разработки

### При добавлении новой функции:
1. Добавить HTML элементы в `index.html`
2. Добавить CSS стили в `styles.css`
3. Добавить JavaScript переменные в начало `app.js`
4. Добавить функции в `app.js`
5. Добавить обработчики событий в конец `app.js`
6. Убедиться, что функция `loadWords()` не нарушена

### Важные правила:
- Не использовать внешние библиотеки/фреймворки
- Весь код в одном файле `app.js`
- Стили в отдельном файле `styles.css`
- Сохранять обратную совместимость с существующими функциями
- Приоритет — чистота кода и читаемость

## 🐛 Частые проблемы и решения

| Проблема | Решение |
|----------|---------|
| Слова не загружаются | Проверить путь к JSON в `loadWords()`, наличие файла в папке `data/` |
| Режим презентации не показывает все слова | Убедиться, что `presentationWordsList = [...words]` (копия массива) |
| Прогресс не сохраняется | Проверить ключ `STORAGE_KEY` = "germanTrainerProgress" |
| Таймер не останавливается | Очистить `presentationTimer` и `presentationAnimationFrame` |
| Кнопки не реагируют | Проверить, что элементы найдены в DOM (код после загрузки) |
| Аудио не работает | Проверить `soundEnabled = true` и поддержку браузером Web Speech API |
| Боковая панель накладывается на кнопки | Проверить `padding-top: 80px` у `.presentation-sidebar` |
| Кнопки в презентации не видны | Убедиться, что используется тёмный фон `rgba(0,0,0,0.7)` |
